# 🚀 Business Finder App

Sistema backend desarrollado con Spring Boot que permite encontrar negocios cercanos sin página web, filtrando automáticamente aquellos con número telefónico disponible en Google Maps.

## 🧠 Funcionalidad

- Búsqueda de negocios por ubicación
- Integración con Google Places API
- Filtrado inteligente:
  - Sin página web
  - Con número telefónico disponible
- Automatización con n8n
- Exportación automática a Google Sheets
- Preparado para integración con IA (Gemini)

## 🛠️ Tecnologías

- Java 17
- Spring Boot
- Maven
- Docker
- n8n
- Google Places API
- Google Sheets API
